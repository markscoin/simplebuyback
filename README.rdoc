== SimpleBuyBack (Under Construction 10/2/14)  

SimpleBuyBack is a website that allows users to recieve quotes for their used electronics.  

The website utilizes the Spree Ecommerce gem for managing products.  
The homepage provides custom navigation making it easier to navigate the "store" (select device).  


Under Development:  
Crawler - Crawls competitor's site to gather prices and products and automatically updating simplebuyback accordingly.
